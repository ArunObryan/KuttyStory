# Kanmani — Color Palette & Design System

## Concept
An interactive storytelling website where a man narrates his love through poetry, music, images, and animations. The palette uses **white**, **blue**, **pink**, and **lavender** shades for a soft, romantic feel.

---

## Color Palette: White, Blue, Pink, Lavender

### White Shades

| Name | Hex | Use |
|------|-----|-----|
| **White** | `#FFFFFF` | Pure white, cards, buttons |
| **Off White** | `#FAFBFC` | Main background |
| **Snow** | `#F8F9FA` | Soft backgrounds |

### Blue Shades

| Name | Hex | Use |
|------|-----|-----|
| **Soft Blue** | `#B8D4E8` | Accents, gradients |
| **Light Blue** | `#D4E8F5` | Soft backgrounds |
| **Sky Blue** | `#A8C8E8` | Highlights |
| **Deep Blue** | `#5B7FA3` | Headings, poetry, buttons |

### Pink Shades

| Name | Hex | Use |
|------|-----|-----|
| **Soft Pink** | `#F5D5E0` | Backgrounds, warmth |
| **Pink** | `#E8B4C4` | Accents, labels, footer |
| **Blush Pink** | `#F0D0D8` | Hover states |

### Lavender Shades

| Name | Hex | Use |
|------|-----|-----|
| **Lavender** | `#E6E0F0` | Borders, sections |
| **Soft Lavender** | `#D8D0E8` | Cards, poetry blocks |
| **Light Lavender** | `#EDE8F5` | Gradients |

### Text & Overlays

| Name | Hex | Use |
|------|-----|-----|
| **Charcoal** | `#4A4A4A` | Body text |
| **Overlay** | `rgba(91, 127, 163, 0.4)` | Image overlays |
